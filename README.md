# Zapret-GUI

![Screenshot 1](https://via.placeholder.com/400x300?text=Zapret-GUI+Main)
![Screenshot 2](https://via.placeholder.com/400x300?text=Zapret-GUI+Dropdown)

Графический интерфейс для управления утилитой `zapret`. Легковесное решение с акцентом на минимализм и чистоту UI.

## Стек технологий
- **Backend:** Rust / Tauri v2
- **Frontend:** React
- **Шрифт:** Montserrat

## Возможности
- **Управление одной кнопкой:** Запуск и остановка службы в один клик.
- **Выбор стратегий:** Быстрое переключение между готовыми конфигурациями обхода.
- **Автоподбор:** Автоматическое тестирование доступных стратегий для поиска рабочей.
- **Минимальный след:** Низкое потребление системных ресурсов благодаря Rust.
- **Чистый UI:** Темная тема в стиле Discord без лишних анимаций.

## Установка

### Требования
- [Rust](https://www.rust-lang.org/tools/install)
- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/installation) (рекомендуется)

### Сборка из исходников
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-repo/zapret-gui.git
   cd zapret-gui
   ```
2. Установите зависимости:
   ```bash
   pnpm install
   ```
3. Соберите приложение:
   ```bash
   pnpm tauri build
   ```
Готовые исполняемые файлы будут находиться в директории `src-tauri/target/release/bundle`.

Готовые сборки также доступны в разделе [Releases](https://github.com/your-repo/zapret-gui/releases).
