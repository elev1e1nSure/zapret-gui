# Zapret GUI

Минималистичный графический интерфейс для управления [zapret](https://github.com/bol-van/zapret) на Windows. Разработано с использованием Tauri, Rust и React.

## Особенности

- **Discord-style UI**: Темная тема, компактный и плотный интерфейс.
- **Быстрое переключение**: Удобный выбор стратегий обхода DPI прямо из приложения.
- **Автономность**: Все необходимые бинарные файлы включены в сборку.
- **Безопасность**: Запуск процессов без лишних консольных окон.
- **Диагностика**: Встроенные средства проверки системы на наличие конфликтующего ПО.

## Использование

1. Выберите подходящую стратегию из выпадающего списка в нижней части окна.
2. Нажмите на центральную кнопку питания для запуска.
3. Синее свечение кнопки означает, что сервис активен.
4. Для остановки нажмите кнопку питания повторно.

## Структура проекта

- `src/`: Фронтенд на React + Vite.
- `src-tauri/`: Бекенд на Rust и конфигурация Tauri.
- `src-tauri/engine/`: Ядро zapret, списки и скрипты стратегий.

## Разработка

Для запуска в режиме разработки:

```bash
pnpm tauri dev
```

Для сборки приложения:

```bash
pnpm tauri build
```
